Project Abstract

The primary aim of my project is to produce a mobile application that teaches people how to edit photos effectively and provides an intuitive interface to practice applying these edits. The research question for this project is: "Is it possible to produce an intuitive yet sophisticated piece of photo editing software from which the user is also able to bolster their photo editing skills?"

I am implementing a range of adjustments from scratch, including a context-aware spot healing tool and bilateral filtering functionality, whilst also researching and applying some of the fundamental principles of effective user experience design. The application is developed in Swift, and also utilises the SwiftUI framework and CIKernel language.
