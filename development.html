Project Goals

  • Produce an intuitive application to allow users to access information surrounding photo editing
  • Provide advice on a range of photo editing topics
  • Allow users to practice making a range of edits without having to leave the tutorial
  • Provide an application that works effectively on mobile devices, and incorporates a range of modern features

    • Project Deliverables

    As part of the planning process, I formulated a list of deliverables for my project:

    • Application
      • Provide an intuitive user interface
      • Produce an application in Swift that's compatible with the current range of iOS devices
      • Base my user interface on the SwiftUI framework
      • Ensure my user interface integrates well with the OS
      • Ensure my user interface works effectively on a range of devices
    • Adjustments
      • Implement a range of image adjustments using the CIKernel language
      • Implement basic adjustments
        • Exposure
        • Contrast
        • Cropping
        • Greyscale
        • Brightness
      • Implement advanced adjustments
        • Noise Reduction
        • Spot Healing
        • Graduated Filter
        • Saturation
        • Haze Reduction
