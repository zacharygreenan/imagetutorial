How I'm Evaluating the Project

Evaluation was performed via ’user testing’, where the testing of the solution is performed by real members of the target audience who are able to use the application themselves and reach their own conclusions.

The process was performed as follows:

  1. Each potential evaluation candidate was asked whether or not they wish to participate
  2. The candidates were asked to complete a pre-session questionnaire designed to gauge their current mobile device and image-editing experience, as well their enthusiasm for bolstering their image-editing skills.
  3. They are asked to independently complete a series of tasks, using the application, designed to cover the entirety of the user interface and functionality.
  4. A post-session questionnaire is completed designed to gauge their opinion of the application.

Ordinarily, this process would take place in person. However, the recent COVID-19 pandemic meant that the evaluation method had to be adapted to remove any in-person contact. Ergo, the following changes were made:

  • Evaluation instructions and consent forms were delivered to each candidate digitally
  • The evaluation took place over video-conferencing software
  • The pre- and post- questionnaire form questions were asked, with the candidate's answer being recorded by me
  • Instead of using the application hands-on, the candidate was able to see a live view of the application via the video conferencing software.
  • In order to complete the series of tasks using the application, the candidate instead was asked to guide me through their thought-process for each task step-by-step.

Evaluation participants were assigned to one of two user-groups:

  1. Users with limited photo-editing experience but good mobiledevice experience
  2. Users with both limited photo-editing and mobile device experience.

Evaluation Results

The results of the evaluation were overwhelmingly positive. Users cited the simple interface, wide range of tutorials, intuitive editing tools and the guided learning process as the main strengths of the application. During the interactive evaluation session, users were all able to complete the tutorials with ease and navigate the interface successfully.

Overall, users rated the application (on average) 8.6 out of a possible 10. The range of scores was between 6 and 10, with the mode score being 9.

Subsequent Improvements

As a result of the evaluation process, several areas of improvement for the application were highlighted. Ergo, the following changes were enacted:

  • The ability for a user to import their own photo to be edited was added
  • The 'Erase All Progress and Favourites' button was relocated to the user profile page, making it easier to find; it was previously in the 'edit profile' page
  • A section as added to the help area detailing the tutorial unlocking mechanism.
  • A tutorial featuring all edits was added
