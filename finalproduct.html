Project Conclusion

At the project's conclusion, the project's goals and deliverables were all fully met and satisfied. The user is able to:

  • Access a wealth of image-editing information and practice making the associated edits alongside
  • Use the application on a mobile device, supporting smartphones and tablets running iOS; it also supports devices running MacOS 10.15+
  • Practice all edits specified in the project's plan, each of which produces output comparable to professional image editing applications.
  • Learn in a guided and engaging manner, due to the incorporation of gamification elements throughout
  • Safely store their progress and synchronise it across multiple devices using the cloud-synchronisation support

The research question for this was: "Is it possible to produce an intuitive yet sophisticated piece of photo editing software from which the user is also able to bolster their photo editing skills?". The solution demonstrates that this is indeed possible.
